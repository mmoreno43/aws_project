About this project

This site is not just an HTML code I created to show off who I am. It is MUCH more than that. I wanted to take everything I had learned within the past few weeks and make something out of it. This site is statically hosted on an S3 bucket in AWS. Eventually, the goal is to move it from S3 to a web server. Within my AWS, I have registered a custom domain to serve my Amazon S3 content. By, default a bucket name is extremely long and I would never remember it. The content within the bucket comes from Travis CI. I had to create a Node.js file that copies my Git master branch and puts the contents into my S3 bucket. My .js file has programmatic access to log into my AWS and perform the move into my bucket. The files come from my laptop where I push the contents to my Git repository. 30 seconds later I see the changes made to my website. As I continue to work through the project, I will add more features. One of them being able to click on the links in this box and having the box slide up to take the place next to "About Me" rather than scrolling me down to that specific box. Another is being able to use Google Analytics to track users coming to my site which causes the site to move from S3 and onto a web server. You can find more about this project here .
